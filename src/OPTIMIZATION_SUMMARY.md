## 🎨 整体设计风格统一

### 色彩系统
- **主色**: 青金色（#14b8a6）
- **辅色**: 暗金色（#f59e0b）
- **背景**: 深板岩（slate-950）
- **次要背景**: 青金暗色（teal-950）
- **文字**: 
  - 主要：青绿色（teal-100/200/300）
  - 次要：青金色半透明（teal-400/60）
  - 强调：暗金色（amber-400/500）

### 材质纹理
- **青玉纹理**: 45度斜纹 overlay（opacity: 5%）
- **金属质感**: 渐变边框 + 内发光
- **模糊玻璃**: backdrop-blur-sm/xl
- **法阵纹理**: SVG 同心圆 + 辐射线

### 动画系统
- **入场**: fade + slide/scale
- **悬停**: scale 1.05
- **点击**: scale 0.9
- **呼吸**: opacity/scale 循环
- **旋转**: 360度循环（20-30秒）
- **脉动**: 缩放循环（1-2秒）

### 阴影层次
- **小**: 0 2px 8px rgba(0,0,0,0.3)
- **中**: 0 4px 20px rgba(0,0,0,0.4)
- **大**: 0 0 30px rgba(20,184,166,0.1)
- **发光**: 0 0 15px rgba(20,184,166,0.2)

### 边框系统
- **细边框**: 1px solid rgba(20,184,166,0.1)
- **标准边框**: 2px solid rgba(20,184,166,0.3)
- **高亮边框**: 2px solid rgba(20,184,166,0.6)
- **渐变边框**: border-gradient（青金→暗金）

### 圆角规范
- **小圆角**: 4px（scrollbar, small elements）
- **标准圆角**: 8px（buttons, badges）
- **大圆角**: 12px（cards）
- **特大圆角**: 16px（modals）
